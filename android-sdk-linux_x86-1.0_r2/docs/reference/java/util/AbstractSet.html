java.util.AbstractSet<E>

java.lang.Object
java.util.AbstractCollection<E> Collection<E>
java.util.AbstractSet<E> Set<E>

AbstractSet is an abstract implementation of the Set interface. This Implementation does not support adding. A subclass must implement the abstract methods iterator() and size().

Known Direct Subclasses
CopyOnWriteArraySet<E>, EnumSet<E extends Enum<E>>, HashSet<E>, TreeSet<E>
CopyOnWriteArraySet<E> A Set that uses CopyOnWriteArrayList for all of its operations. 
EnumSet<E extends Enum<E>>  
HashSet<E> HashSet is an implementation of Set. 
TreeSet<E> TreeSet is an implementation of SortedSet. 
Known Indirect Subclasses
LinkedHashSet<E>
LinkedHashSet<E> LinkedHashSet is a variant on HashSet.
